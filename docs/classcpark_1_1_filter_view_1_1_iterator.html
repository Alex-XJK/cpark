Detailed Description

template<std::ranges::view V, typename Func>
class cpark::FilterView< V, Func >::Iterator

A lazily evaluated iterator that can filter values from some original iterator. Because std::ranges::filter_view does not provide begin() const and end() const, we can not safely rely on it to implement FilterRdd without breaking const-correctness. So we define our own filter iterator here.
