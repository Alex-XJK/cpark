class cpark::FilterRdd< R, Func >

An Rdd holding the data filtered from an old rdd by some function.

Template Parameters
RType of the old Rdd.
FuncType of the transformation function. The type of the old Rdd R's elements should be able to invoke function Func, and the type shouldn't change after filter.
