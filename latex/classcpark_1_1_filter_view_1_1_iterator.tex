class cpark\+::\+Filter\+View$<$ V, Func $>$\+::\+Iterator}
A lazily evaluated iterator that can filter values from some original iterator. Because std\+::ranges\+::filter\+\_\+view does not provide {\ttfamily \doxylink{classcpark_1_1_filter_view_a6d54e309bc10d85cc41669bbb788dc98}{begin() const}} and {\ttfamily \doxylink{classcpark_1_1_filter_view_af52ebb94634dd9d42b0b45ba1013a539}{end() const}}, we can not safely rely on it to implement \doxylink{classcpark_1_1_filter_rdd}{Filter\+Rdd} without breaking const-\/correctness. So we define our own filter iterator here.
