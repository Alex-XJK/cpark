\doxysubsection{Detailed Description}
A concept that requires its objects have some dependency relationship between each other. A {\ttfamily Has\+Dependency} must be {\ttfamily Has\+Id}, with the following member functions\+: dependencies() returns a list of the id-\/s of the dependencies of the current object. add\+Dependency() adds the id of a new dependency to the current object.
